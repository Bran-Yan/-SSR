### 2.2 软件操作和效果
由于软件禁止截屏录屏，所以采用拍照的形式，效果欠佳。

a.**打开**SSRR软件
![1](./img/手机端使用1.png)



b.点击**框框**位置，选择节点
![2](./img/手机端使用2.png)



c.点击"**闪电**"图标就可以测速，查看节点是否可用
![3](./img/手机端使用3.png)



d.点击"**+**",点击"**添加/升级SSR订阅**"
![4](./img/手机端使用4.png)



e.点击"**添加订阅地址**"
![5](./img/手机端使用5.png)



f.把**SSR地址**粘贴到输入框 (友情提供一个https://raw.githubusercontent.com/ssrsub/ssr/master/ssrsub)
![6](./img/手机端使用6.png)



g.点击**飞机图标**连接，点击**框框**测速
![7](./img/手机端使用7.png)



h.打开<https://www.youtube.com/>效果
![8](./img/手机端使用8.png)



**i.完成！**
